## ShellFish

| Action       | Command |
|--------------|---------|
| New Terminal | ⌘T      |
| Cancel       | ⌘.      |
| Connect      | ⏎       |
| Edit Server  | ⤃       |
| Enable tmux  | ⌥⌘T     |

## Terminal

| Action          | Command |
|-----------------|---------|
| Quick Terminal  | ⌥⌘N     |
| Close Terminal  | ⌘W      |
| New Terminal    | ⌘T      |
| Port Forwarding | ^⌘F     |
